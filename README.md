## Краткое описание работы
В ходе данной лабораторной работы было создано веб-приложение «Список задач» (TODO List). Приложение позволяет пользователю добавлять новые задачи, отмечать их как выполненные и удалять. Реализация выполнена с использованием чистого JavaScript (Vanilla JS), HTML и CSS. Также была настроена интеграция с системой контроля версий Git и удалённым репозиторием GitHub.

## Описание (что изучили)
В процессе выполнения работы были закреплены следующие навыки:
- Работа с DOM-деревом: поиск элементов, создание новых узлов, манипуляция классами.
- Обработка событий пользователя (клики, нажатие клавиш).
- Работа с формами и полями ввода.
- Динамическое обновление интерфейса без перезагрузки страницы.
- Инициализация и работа с Git-репозиторием (init, add, commit, push, remote).
- Структурирование проекта (разделение на HTML, CSS, JS).
